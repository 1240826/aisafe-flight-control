# SCOMP - Computer Systems Component

This folder is part of the project repository setup (US003) and will contain the C language components to be developed for the SCOMP (Computer Systems) course unit in upcoming sprints.

At this stage, the folder exists to establish the repository structure and make clear the separation between the Java-based system (`aisafe.base/`) and the C-based simulation component.

## Notes

- This component is not part of the Maven build.
- Development will take place on a Linux virtual machine.
- Source code and documentation will be added here as the relevant user stories are developed.
