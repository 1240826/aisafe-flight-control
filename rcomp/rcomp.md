# RCOMP - Computer Networks Component

This folder is part of the project repository setup (US003) and will contain the network-related components to be developed for the RCOMP (Computer Networks) course unit in upcoming sprints.

At this stage, the folder exists to establish the repository structure and make clear the separation between the Java-based system (`aisafe.base/`) and the network infrastructure component.

## Notes

- This component is not part of the Maven build.
- Network protocol specifications, configurations, and any client applications will be added here as the relevant user stories are developed.
